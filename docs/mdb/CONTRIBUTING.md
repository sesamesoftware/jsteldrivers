# Contributing

This repository is an archived, read-only release of the Sesame Software MDB JDBC Driver. It is not under active development and pull requests are not being reviewed or merged.

## Forking

You are welcome to fork this repository and maintain your own version under the terms of the [Apache 2.0 License](../../LICENSE). The license permits modification and redistribution, with or without changes, provided the attribution and license notices are retained.

## Questions and usage support

For questions about integration or usage, please contact [Sesame Software](https://www.sesamesoftware.com).

## Security concerns

Known security advisories for bundled dependencies are documented in [SECURITY.md](../SECURITY.md). This repository is not actively monitored for new vulnerability reports.
